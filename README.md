# hello-angular-azdo

A minimal Angular project to test CI execution on GitHub hosted runners and integration with Azure DevOps.

The app displays a simple page with the text:  
**"Hello World from Angular + GitHub + PR CI test"**

## Prerequisites

- [Node.js](https://nodejs.org/) v20 or later
- [npm](https://www.npmjs.com/) (comes with Node.js)

## Run locally

```bash
npm install
npm start
```

Then open your browser at `http://localhost:4200`.

## Build the project

```bash
npm install
npm run build
```

The production build artifacts will be placed in the `dist/` directory.
